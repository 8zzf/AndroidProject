# AndroidProject
###项目技术描述
  1)Android5.0 新控件的使用:如RecyclerView,CardView,ToolBar等
  
  2）Rxjava +Retrofit+Lrucache+DiskLrucache完成三级缓存
  
  3）面向holder编写控件
  
  4）自定义控件：如 妹子浏览控件，圆形控件，boomMenu,高斯模糊等
  
  5）第三方库：GSON,Butterknife等等
